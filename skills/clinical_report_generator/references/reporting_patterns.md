# Clinical Report Patterns

Use these report shapes for the shared HTML/PDF generator:

## Recommended structure

1. Title
2. Subtitle
3. One-sentence summary
4. Metric strip
5. Visual summary
6. Report sections
7. Footer or next-step prompt

## Preferred sections

- Signal
- Evidence
- Actions
- Review

## Formatting rules

- Keep each section short.
- Use plain clinical language.
- Show numeric change when available.
- Keep any optional PDF output as a fallback, not a dependency.
