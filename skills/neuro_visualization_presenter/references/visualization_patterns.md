# Neuro Visualization Patterns

Use these patterns when packaging visuals:

## Recommended payload

- `slice_preview`
- `overlay_preview`
- `trend`
- `timeline`
- `comparison`
- `viewer_manifest`

## Presentation goals

- Make the primary image obvious.
- Keep the timeline light.
- Put the quantitative trend next to the preview, not below a large text wall.
- Use the manifest to support later NiiVue integration.

## Output guidance

- Prefer compact labels.
- Use a single baseline / latest comparison if space is limited.
- Avoid dumping raw paths into the chat surface.
